### Syntax ###

<div class="highlight irssisyntax"><pre style="\-\-cmdlen:2ch"><code><span class="synB">CD</span> <span class="synB05">&lt;directory></span></code></pre></div>



### Parameters ###

The directory to change into.

### Description ###

Changes the current active directory.

### Examples ###

    /CD /home/public_ftp
    /CD /home/mike
    /CD /var/log
